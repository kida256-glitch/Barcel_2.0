# **App Name**: CeloBargain

## Core Features:

- Wallet Connection: Enable wallet connection via Celo blockchain.
- Seller Product Listing: Allow sellers to list products with prices, images, profile, bio, and success rate.
- Seller Price Offers: Enable sellers to specify different price options for a product.
- Buyer Price Selection: Allow buyers to select from seller's prices or propose a custom price.
- Transaction Approval: Provide approval button for both buyer and seller to initiate the transaction.
- Product Purchase: Facilitate transaction on Celo network when both parties approve.
- Buyer Rating and Comments: Enable buyers to rate and leave comments for sellers after a transaction.  A generative AI tool can be used to flag negative comments, or generate a TL;DR version.

## Style Guidelines:

- Background color: Dark charcoal (#121212) for a sleek black theme.
- Primary color: Electric purple (#BE52F2) to give a modern and vibrant feel.
- Accent color: Cyan (#48D1CC) for interactive elements and highlights.
- Body and headline font: 'Space Grotesk', a modern sans-serif font for a contemporary feel.
- Use minimalist, outlined icons in cyan (#48D1CC) to maintain a clean interface.
- Implement a grid-based layout for structured content presentation.
- Subtle transitions and animations for UI elements to enhance user experience.
